# Этап 2.2: Обновить Helper Tools и Ping с метаданными

**Цель:** Добавить category, subcategory, priority, tags в вспомогательные инструменты

**Затронутые файлы:** 3 tools

**Время:** 0.5-1 час

**Зависимости:** Требует завершения этапа 1.1

**Параллельность:** Можно выполнять параллельно с этапом 2.1

---

## Категоризация Helper Tools

| Tool | Category | Subcategory | Priority | Tags |
|------|----------|-------------|----------|------|
| `PingTool` | `system` | `health` | `normal` | `["ping", "health", "status"]` |
| `IssueUrlTool` | `helpers` | `url` | `normal` | `["url", "link", "helper"]` |
| `DemoTool` | `helpers` | `demo` | `low` | `["demo", "example", "test"]` |

---

## Чек-лист задач

### 1. PingTool

**Файл:** `packages/servers/yandex-tracker/src/tools/ping.tool.ts`

- [ ] Обновить `METADATA.category = 'system'`
- [ ] Обновить `METADATA.subcategory = 'health'`
- [ ] Обновить `METADATA.priority = 'normal'`
- [ ] Обновить `METADATA.tags = ['ping', 'health', 'status']`
- [ ] Обновить `description`: добавить префикс `[System/Health]`

**Пример description:**
```typescript
description: '[System/Health] Проверка доступности сервера'
```

### 2. IssueUrlTool

**Файл:** `packages/servers/yandex-tracker/src/tools/helpers/issue-url/issue-url.tool.ts`

- [ ] Обновить `METADATA.category = 'helpers'`
- [ ] Обновить `METADATA.subcategory = 'url'`
- [ ] Обновить `METADATA.priority = 'normal'`
- [ ] Обновить `METADATA.tags = ['url', 'link', 'helper']`
- [ ] Обновить `description`: добавить префикс `[Helpers/URL]`

**Пример description:**
```typescript
description: '[Helpers/URL] Получить URL задачи в веб-интерфейсе'
```

### 3. DemoTool

**Файл:** `packages/servers/yandex-tracker/src/tools/helpers/demo/demo.tool.ts`

- [ ] Обновить `METADATA.category = 'helpers'`
- [ ] Обновить `METADATA.subcategory = 'demo'`
- [ ] Обновить `METADATA.priority = 'low'`
- [ ] Обновить `METADATA.tags = ['demo', 'example', 'test']`
- [ ] Обновить `description`: добавить префикс `[Helpers/Demo]`

**Пример description:**
```typescript
description: '[Helpers/Demo] Демонстрационный инструмент для тестирования'
```

**Примечание:** DemoTool имеет `priority: 'low'` — будет в конце списка.

### 4. Валидация

- [ ] Запустить `npm run typecheck --workspace=mcp-server-yandex-tracker`
- [ ] Запустить `npm run build --workspace=mcp-server-yandex-tracker`
- [ ] Все ошибки исправлены
