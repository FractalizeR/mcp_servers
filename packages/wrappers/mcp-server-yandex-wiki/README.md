# mcp-server-yandex-wiki

This is a convenience wrapper for [`@fractalizer/mcp-server-yandex-wiki`](https://www.npmjs.com/package/@fractalizer/mcp-server-yandex-wiki).

## Installation

```bash
npm install -g mcp-server-yandex-wiki
# or
npm install -g @fractalizer/mcp-server-yandex-wiki
```

For full documentation, see the [main package](https://github.com/FractalizeR/mcp_servers/tree/main/packages/servers/yandex-wiki).
