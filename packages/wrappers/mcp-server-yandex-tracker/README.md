# mcp-server-yandex-tracker

This is a convenience wrapper for [`@fractalizer/mcp-server-yandex-tracker`](https://www.npmjs.com/package/@fractalizer/mcp-server-yandex-tracker).

## Installation

```bash
npm install -g mcp-server-yandex-tracker
# or
npm install -g @fractalizer/mcp-server-yandex-tracker
```

For full documentation, see the [main package](https://github.com/FractalizeR/mcp_servers/tree/main/packages/servers/yandex-tracker).
