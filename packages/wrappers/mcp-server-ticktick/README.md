# mcp-server-ticktick

This is a convenience wrapper for [`@fractalizer/mcp-server-ticktick`](https://www.npmjs.com/package/@fractalizer/mcp-server-ticktick).

## Installation

```bash
npm install -g mcp-server-ticktick
# or
npm install -g @fractalizer/mcp-server-ticktick
```

For full documentation, see the [main package](https://github.com/FractalizeR/mcp_servers/tree/main/packages/servers/ticktick).
